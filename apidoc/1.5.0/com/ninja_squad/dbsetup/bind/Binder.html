com.ninja_squad.dbsetup.bind
Interface Binder

public interface Binder

An object which binds a value to a prepared statement parameter. It's advised to make implementations of this interface immutable, and to make them implement equals and hashCode in order for DbSetupTracker to function properly, or to make them singletons.
